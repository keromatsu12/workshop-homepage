## 8. 課題・改善候補

| #   | 区分             | 内容                                                                                                                                        |
| --- | ---------------- | ------------------------------------------------------------------------------------------------------------------------------------------- |
| 1   | アクセシビリティ | `<header>` に `role="banner"` は暗黙的に付与されるが、`<main id="main-content">` に `role="main"` の明示的付与を検討できる（任意）          |
| 2   | アクセシビリティ | ハンバーガーメニューなどモバイル向けナビゲーションの切り替えUIが未実装。800px以下でナビが横スクロールとなるため、タップ操作性の改善余地あり |
| 3   | SEO / OGP        | `og:url`、`og:image` などの OGP タグが未設定。SNS シェア時のプレビュー画像を設定することを推奨                                              |
| 4   | パフォーマンス   | `<link rel="preconnect">` や `<link rel="preload">` の設定なし（今回は外部フォント未使用のため影響軽微）                                    |
| 5   | インタラクション | ナビリンクのアクティブ状態（スクロール位置に応じて現在セクションのリンクをハイライト）が未実装                                              |
| 6   | アクセシビリティ | `<p class="site-header__brand">` は見た目上ブランド名だが `<a>` や `<strong>` ではない。トップへのリンク化を検討できる                      |
